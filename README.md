# Thrive and Prosper private limited Website

A React.js website for Thrive and Prosper private limited product.

## Installation

1. Extract the project files
2. Open terminal in the project directory
3. Run: `npm install`
4. Run: `npm start`

## Available Scripts

- `npm start` - Runs the app in development mode
- `npm build` - Builds the app for production
- `npm test` - Launches the test runner

## Customization

You can easily customize the content, colors, and styling by editing the component files and App.css.
